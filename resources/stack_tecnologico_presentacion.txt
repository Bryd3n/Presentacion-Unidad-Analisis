STACK TECNOLÓGICO PARA PRESENTACIÓN ANIMADA (FarmaHumana)

El siguiente stack ha sido seleccionado para desarrollar una presentación de producto animada, profesional y altamente personalizada, ideal para integrar dentro de un ecosistema web:

1. ESTRUCTURA Y MAQUETACIÓN
   - HTML5: Base semántica para la estructura de las diapositivas.
   - Tailwind CSS: Framework de utilidad para el diseño responsivo, alineado con la guía de estilos (Helvetica Neue, colores de marca #F6FFFF, #50E9F8, #00006A, #179150).

2. ANIMACIÓN Y MOVIMIENTO
   - GSAP (GreenSock Animation Platform): Motor principal para animaciones de alto rendimiento y control de líneas de tiempo.
   - ScrollTrigger (Plugin de GSAP): Utilizado para disparar animaciones basadas en el avance del usuario (scroll o navegación por diapositivas).

3. CONTROL DE NAVEGACIÓN
   - Reveal.js: Framework especializado para la gestión de slides, transiciones de página, atajos de teclado y modo presentador.

4. INTEGRACIÓN
   - JavaScript (ES6+): Lógica para conectar los eventos de cambio de slide de Reveal.js con las animaciones de GSAP.

NOTAS TÉCNICAS:
- Este stack permite mantener la identidad visual definida en la "Guía de estilos Farmahumana - Semana 1".
- Se recomienda el uso de variables CSS para centralizar la paleta de colores corporativa.
